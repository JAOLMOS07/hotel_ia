🔬 Arquitectura del Sistema

📥 Fuentes de datos:

  • DANE-EMA: Series históricas ocupación 2007–2025
  • EGIT-DANE 2023/2024: Microdatos perfil turista (133 vars)
  • RNT-Valledupar: 328 establecimientos activos
  • Cámara Comercio Valledupar: Impacto Festival
  • Cotelco Cesar: Estadísticas ocupación regional

🤖 Modelos implementados:

  • LSTM: 2 capas (64→32 neuronas) + Dropout 0.2 + BatchNorm
  • K-Means: K=5 | EarlyStopping | Silhouette 0.62
  • Recomendación: Híbrido (45% colaborativo + 55% contenido)

🖥 Stack tecnológico:

  • Backend: Python + FastAPI + TensorFlow + scikit-learn
  • Frontend: HTML5 + Chart.js
  • Base de datos: PostgreSQL

📊 Comparación de Métricas por Modelo

Estado del sistema: Ejecute python main.py en la carpeta sistema_turistico/ para entrenar los modelos y generar todos los resultados. Luego inicie la API con uvicorn api.main:app --reload --port 8000.
